## Installation
You can use [Composer](https://getcomposer.org/) for the most of applications.

```bash
composer require iaretedd/http
composer install
```
...or just take classes you need from the `src` directory. 

## Tests
To run PHP Unit tests:
```bash
./vendor/bin/phpunit
```
